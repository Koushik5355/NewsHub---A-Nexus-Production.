📰 NewsHub — A Nexus Production

NewsHub is a dynamic web application that aggregates the latest news from multiple trusted sources such as BBC, CNET, The Verge, and Business Insider — all in one place.

It provides users with real-time headlines, images, and direct article links without relying on any database or cloud storage.

🚀 Features
🔎 Live News Aggregation from multiple popular platforms
🖼️ Rich Content Extraction (Headlines, Images, Article Links)
⚡ Lightweight & Fast — No database dependency
🎨 Clean UI using Bootstrap templates
🔄 Fresh Data on Every Load
🛠️ Tech Stack

Frontend :
HTML
CSS
JavaScript
Bootstrap

Backend :
Python
Flask

Data Extraction :

Beautiful Soup
NewsAPI

⚙️ How It Works :
The backend fetches news data from external sources using NewsAPI
Beautiful Soup parses HTML content to extract:
Headlines
Images
Article links
The frontend dynamically displays the fetched content
Every page load triggers fresh scraping → ensuring up-to-date news

💡 Key Highlights :
❌ No Database Required

🧩 Modular & Scalable Design
🔍 Category-based filtering (Tech, Business, Sports, etc.)
📱 Responsive enhancements for mobile devices
